# ⚔️ Solo Quest (solo-quest-web)

### 🎯 What is it?
Solo Quest is an AI-powered task management and productivity web application. It turns your daily tasks into "quests," gamifying your routine with the help of Google's Generative AI. Built with React and Vite, it offers a seamless and responsive user experience. 

### ✨ Features
- **AI-Powered Quest Generation**: Uses Gemini API to break down large goals into actionable, bite-sized quests.
- **Local First Architecture**: Uses Dexie (IndexedDB) for fast, offline-capable local storage of your tasks.
- **Modern UI & Gamification**: Smooth animations with Framer Motion and a clean interface built with Tailwind CSS.
